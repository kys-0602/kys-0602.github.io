## 7월

싸피 14기 면접에서 개같이 탈락한 후, 집에서 메이플 스토리를 하던 중 02-3429-\*\*\*\*(1)로 전화가 왔다. 서울캠퍼스에 추가 합격되었으니, 오늘 안으로 증명 사진을 제출해달라는 것이었다. 싸피의 추가 합격은 내 인생에서 아직도 실감이 잘 안 나는 상황 중 하나다.
{ .annotate }

1.  아직도 생각나는 번호

싸피는 입과 후 바로 수업을 진행하지 않고, 아이스 브레이킹을 위한(?) 스타트캠프라는 것을 진행한다. 본격적으로 싸피 내 시스템에 적응하고, 친구들과 친해져야 하는 그런 프로그램이다. 나는 이걸 참여하고 싶었지만, 추가 합격자는 어림도 없이 스타트캠프 마지막 날에 OT를 한다고 했다.

정확히 7월 18일에 추가 합격자 오리엔테이션을 진행했다. 당시 너무 일찍 간 것 같긴한데, 오전 7시 40~50분 사이에 도착했다. 강의실의 불은 꺼져있고, 해당 층에는 나만 온 듯 싶었다. (지금 생각하면 내 담당 강사님 그 옆에 계셨을 듯)

그리고 한 5분 후에 다른 추가 합격자분이 오셨다. 수원인지, 성남인지 기억이 잘 안 나지만 꽤 일찍 오셔서 놀랬고 다른 분들이 오기 전에 수다를 좀 했다.

오리엔테이션은 하루종일 영상 시청만 하다가 끝났고, 별로 특별한 건 없었다. 점심 식사 전에 분반 테스트를 본 걸로 기억하는데, JAVA에 대해 잘 알고 있는 교육생이라면 손쉽게 풀 수 있는 그런 문제였다. 당시에는 내 주 언어가 C++이었고, JAVA 개념에 익숙하지 않았기 때문에 은근 헷갈리고 어려웠다. 😂

반 배치는 바로 나왔고 8반이라는 결과가 나왔다. 추가 합격자들 사이에서 서로 몇 반인지 물어봤는데, 나랑 같이 가는 분이 1명 있어서 내심 너무 좋았다. 당시에 그분도 엄청 좋아했음. ㅋㅋㅋㅋㅋㅋㅋㅋㅋ. 그리고 지금 생각하니 서로 인사라도 하고 헤어졌어야 했는데 아쉽다.

8반에 처음 입장했을 때, 다들 쳐다보는 눈빛...(1) 그리고 누가한 말인지 모르겠다만 **어?**와 **뭐야?**란 단어가 아직도 생각난다.
{ .annotate }

1.  그래서 일부러 안 둘러봄...

반에 입장하고 나서 인솔해주셨던 프로님은 사라지셨고, 반에 계신 강사님께서 자리 설명 및 짝궁(페어)을 알려주셨다. 이때 짝궁은 나랑 같이 오신 추가 합격자분.

내 옆에 앉아 있던 분은 운이 좋게도(?) 반장이었고, 7월에 있어 싸피에 적응하는 데 큰 도움을 많이 받았다. 프로그램 설치와 사용법부터 해서 은근 귀찮게 한 듯.

7월 말에는 첫 월말 및 과목 평가와 싸피데이라는 걸 진행했다. 뭐 다행이 과락은 면했고(60점 이상), 싸피데이는 흔히 블로그 후기에 보이는 각 반에 있어여 하는(?) 규칙같은 걸 작성했다.



!!! note "입학식 참석"

    이걸 보시는 분이 있다면, 미리 JAVA 공부하는 걸 추천합니다.

## 8월

비전공반에 있던 동갑내기 친구가 우리 전공반으로 월반을 했다. 완전 무관한 전공이던데, 전공자보다 훨씬 나은 성적을 보여서 대단한 친구라고 생각 중이다.

그리고 전공자반이라면 반드시 취득해야 하는 **A형 모의 역량테스트**가 있던 달이기도 했다. 문제는 유출했다간 ㅈ되니 알려줄 수 없지만, 유형은 그리디였다.

분명 문제에서 주어진 TC의 답이 도출되는 게 보였기 때문에 그대로 제출했는데 개같이 탈락했다. 이때부터 저주가 시작된 것 같다.

뭐... 특별한 건 없던 달인 것 같다.

!!! note "A형 모의 역량테스트"
    2시간 안에 2문제 중 1문제를 반드시 해결해야 한다.

    싸피 입과 코딩 테스트를 본 사람은 알겠지만, 정답과는 상관없이 컴파일과 제출 성공 시 Accept으로 나타난다. 즉, 샘플 인풋의 TC가 정답을 도출하는 지 반드시 확인해야 한다.

    ----------

    1문제를 해결하면 A, 2문제 해결 시 A+ 등급이다. 전공자는 A 등급 이상만 받으면 수료 기준에 충족한다.

!!! note "알고리즘 공부"
    JAVA 수업의 진도가 끝나면 알고리즘 공부가 진행된다.

    사실 나는 알고리즘을 아예 모르는 상태로 싸피 입과 코딩 테스트를 보고, 운 좋게 합격했다.(1) 즉, A형 1차 시험 탈락인 건 당연한 것.
    { .annotate }

    1.  싸피 입과 코딩 테스트에서 2문제를 모두 해결한 경우, 자동으로 A+ 등급 취득이고, 입과 후 얼마 안 지나서 B형을 볼 수 있다는 안내가 나타난다. 이게 안 나타난다면 IM 등급이고 0~1솔이라는 것.

    기술 지원과 QA 분야로 직무를 준비했었기 때문에, 전공자임에도 불구하고 알고리즘 공부를 내팽겨쳤었다. 유일하게 알던 알고리즘이 누적합 정도...

    싸피에서 알고리즘 수업의 진도를 나가긴 하지만, 초심자가 듣고 따라가기에는 많이 벅차다. 어느정도 수박 겉핥기 한 사람만 이해할 수 있을 듯 싶다. 미리 말하지만, 저처럼 고통 받기 싫으신 분은 알고리즘 공부 정말 열심히 해오세요.

## 9월

A형 모의 역량테스트 2차가 있었던 달이다. 문제는 알려줄 수 없고, BFS와 DP를 사용해여 하는 문제가 나왔었다.

지금 생각해보면 BFS 문제에 몰빵했다면 풀 수 있었을 것 같던 문제였고, DP 문제는 충격 그 자체였다. DP가 아닌 순수 구현 및 시뮬레이션으로 접근했는데, 시간 초과가 나고 답이 하나씩 틀리는 것이었다. 아무튼 2차도 개같이 탈락했다. 그리고 이때는 오후 6시 이후에 남아서 시험을 치루기 때문에 소소한 간식도 챙겨줬다.

그리고 9월 과목평가에서 50점으로 과락을 받았다. 이땐 ㅈ됐다는 생각보다는 그냥 한 번쯤 맞아보고 싶었는데 잘 됐다였다. 과락 시 재시험의 기회를 주는 회차가 있는데, 다행이었다. 재시험도 역량 테스트와 마찬가지로 오후 6시 이후에 남아서 치룬다.

과목평가 재시험 때에도 ㅈ된 상황이 발생했는데, 이야 이건 풀었다 싶어서 제출하고 나왔는데 출력 형식을 제대로 안 지키고 나왔다. 즉, 출력 형식에 맞춰 출력해야 하는데 그걸 안 지킨 것 ㅋㅋㅋㅋㅋㅋㅋㅋㅋㅋㅋㅋ. 그래서 강사님께 실망시켜드려 죄송하다는 MM을 드리기도 했다.(1)
{ .annotate }

1.  하지만 짜잔?! 운이 좋게도 그 뒤에 있던 문제에서 정답을 출력한 일부 TC가 있어, 부분점수를 얻어 60점으로 간신히 재평가의 과락을 면했다.

## 10월

10월부터는 추석이 있어 금방 지나갔다. 그리고 이번 기수부터 새롭게 추가된 AI 강의가 한 달 동안 진행됐다.

AI 강의는 개인적으로 혹평을 주고 싶다. 전공자라고 해서 AI를 다 아는 사람만 있는 것은 아닐텐데, 이정도의 난이도가 맞나 싶었다.

내가 생각했던 AI 강의는 진짜 완전 AI란 무엇인가부터 해서, 파이썬으로 실습해보고 익히는 걸 생각했는데 전혀 아니었다. 물론 실습도 있긴했는데, 도움이 되는 것도 아니었다.

한 달 동안 진행하는 것이었기 때문에 일부러 그런 것 같긴한데, 조금 더 개편해서 내놓는 게 낫지 않은 가 싶다. 어른의 사정으로 급조한 티가 보였다.

그리고 A형 모의 역량테스트 3차가 진행되었다. 결과부터 말하자면 아깝게(?) 탈락. 문제 유형은 누적합(또는 DP 가능)과 DFS였다. 2번보다는 1번의 문제가 더 해결하기 쉬워보였기에 도전했다가 해결하지 못했다. 시간복잡도 $O(N^2)$으로 해결했다간 시간 초과가 날 듯 싶어서 DP로 접근했는 데 오히려 이러면 안 되는 것이었다. 나중에 보니 스무스하게 통과한다고 했다. 계속 DP 방식으로 해결하려고 점화식 찾는 것에 시간을 너무 썼다.

!!! note "A형 모의 역량테스트"
    싸피에서 A형 모의 역량테스트의 기회를 최대한 많이 제공해주는 것 같습니다. 어떻게든 2학기 보내주려고 하는 게 보입니다. 다만, 기수에 따라 차이가 있을 수 있습니다.

    그리고 대부분의 전공자는 A형 모의 역량테스트를 1차에서 합격하니, 스트레스 및 창피 받기 싫다면 진짜 공부 열심히 해주세요...

!!! note "밋업데이"
    싸피의 1학기 과정을 마무리하는 MeetUp Day 행사가 열립니다. 보통은 11~12월에 하는 것 같은데 생각보다 빠른 날짜에 진행되었습니다.

    초대 가수 부른 후기도 있던데, 이번 기수는 없었습니다. 다른 층에 가서 앉아 있는 시간만 있어가지구 다시 아쉬웠던 행사였습니다. 하지만, 싸피 다음 기수(?) 굿즈 응모했는데, 후보작이 선정되어서 배민 포인트 쿠폰 달달하게 받았습니다.

## 11월

11월도 별로 특별한 건 없는 것 같다. 프론트-엔드로, Vue.JS를 배운 걸로 기억하는데, 생각보다 너무 재밌었고 내 취향에 맞았다.

## 12월

12월은 15기의 면접과 채용박람회가 있기 때문에 3주간 비대면 수업을 진행한다.

비대면은 웹엑스를 통해 진행하기 때문에 본인의 모습이 웹캠에 나와야한다. 이때가 싸피를 다니면서 제일 편하고 행복했다. 교통비가 안 나가니 배달도 마음껏... 잠시나마 코로나 기수가 부러워지는 순간이었다.

12월은 다음 학기를 가기 전 달이기 때문에 마지막 A형 모의 역량테스트가 열렸다. 이때는 운 좋게 문제를 풀 수 있었고, 다음 학기 가는 것이 사실상 확정되었다! 나중에 안 사실인데 이때도 풀지 못해서 퇴소 당한 전공자와 비전공자가 있다고 한다.

그리고 1학기의 꽃이자 고통(?) **관통 PJT**를 진행해야 한다. 지금까지 배운 기술 스택(Java, Spring, MySQL, Vue.JS)을 통해 주어진 명세서와 컨셉을 추가해 구현해야 한다. 앞에서 언급했듯이 나는 기술 지원과 QA 업무를 주로 준비했기 때문에 웹에 대한 기반 지식이 매우 부족했다. 그래서 같은 페어 형한테 백-엔드를 담당해달라고 부탁했다. 프론트-엔드가 그나마 쉽고, 이쪽에 관심이 꽤 있었기 때문에 그랬다.(1)
